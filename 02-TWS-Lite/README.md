# The Wone: Shadows — Prototype

*English | [Tiếng Việt](README.vi.md)*

An early playable prototype of an original action RPG, built solo in 
Unity/C#. This snapshot represents an early stage of development — 
not the current version of the project.

## Concept

Set in the world of Sorath, the game centers on shadow summoning and 
army management: the player collects and commands a controlled army 
of shadows, bound by clear rules around collection limits and summon 
mechanics. The world is built with a lore-driven setting and original 
naming, drawing on Vietnamese cultural identity.

## What's in this prototype

- Player movement and core combat loop
- Basic enemy AI
- Zone-based world progression
- Foundational systems for the shadow summoning mechanic
